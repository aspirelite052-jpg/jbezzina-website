# Company logo

Place your logo file in this folder:

| File | Use |
|------|-----|
| **`logo.png`** | Recommended — transparent or white background, ~400×120px (or similar wide ratio) |
| `logo.svg` | Optional vector version (update `CompanyLogo` if you prefer SVG first) |

Until `logo.png` is added, the header shows a temporary industrial icon mark.

**Tips**
- Keep the mark readable at ~44px height (mobile header).
- Avoid very thin lines; they disappear on small screens.
- Dark/navy logos work on the white header; provide a light variant if you add a dark header later.
